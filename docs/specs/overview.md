# 機能仕様

## 画面一覧

| パス | 画面名 | 説明 |
|---|---|---|
| `/sign-in` | サインイン | Clerk 提供のサインイン UI |
| `/sign-up` | サインアップ | Clerk 提供のサインアップ UI |
| `/(main)/profile-setup` | プロフィール設定 | 初回登録フォーム |
| `/(main)/...` | メイン画面群 | 家計簿機能（別途定義） |

## API

Hono + `@hono/zod-openapi` で構築。`apps/web/app/api/[...route]/route.ts` でマウント。  
認証は `@hono/clerk-auth` ミドルウェアで保護。

## コード値定義

### 性別コード（GENDER_CODE）

| コード | 表示名 |
|---|---|
| 0 | 不明 |
| 1 | 男性 |
| 2 | 女性 |
| 9 | 適用不能 |

### 続柄コード（RELATIONSHIP_CODE）

| コード | 表示名 |
|---|---|
| 1 | 本人 |
| 2 | 配偶者 |
| 3 | 子 |
| 4 | 親 |
| 5 | その他 |

### カテゴリタイプ（CATEGORY_TYPE）

| コード | 表示名 |
|---|---|
| 1 | 支出 |
| 2 | 収入 |

### AI 機能コード（FEATURE_CODE）

| コード | 機能 |
|---|---|
| 1 | レシート読み取り・自動入力 |
| 2 | 支出分析・アドバイス |

### 都道府県コード（REGION_CODE）

1〜47 の整数で表現。詳細は `packages/common/src/ui-constant.ts` の `REGION_MAP` を参照。

## バリデーション規則

| フィールド | 規則 |
|---|---|
| カテゴリ名 | 最大 50 文字 |
| メモ | 最大 255 文字 |
| 生年 | 西暦（整数） |
| 居住地 | REGION_MAP のコード値（1〜47） |
