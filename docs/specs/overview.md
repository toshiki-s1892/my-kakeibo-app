## API

Hono + `@hono/zod-openapi` で構築。`apps/web/app/api/[...route]/route.ts` で全ルートをマウント。

| エンドポイント | 説明                                    |
| -------------- | --------------------------------------- |
| `/api/doc`     | OpenAPI スペック（JSON）                |
| `/api/ui`      | Swagger UI（APIドキュメント・動作確認） |

### 認証

`clerkMiddleware()` をメインappでグローバルに適用。`/api/doc`・`/api/ui` のみ認証不要。

### ルート構成

各機能ごとに `server/routes/{feature名}/` ディレクトリを作成し、以下の3ファイルで構成する。

| ファイル     | 役割                     |
| ------------ | ------------------------ |
| `schema.ts`  | Zodスキーマ・OpenAPI定義 |
| `handler.ts` | DBアクセスなどの処理     |
| `index.ts`   | ルートを登録してexport   |

### エラーレスポンス

全ルート共通のスキーマを使用する。

```json
{
  "message": "バリデーションエラー",
  "details": [{ "field": "displayName", "message": "50文字以内で入力してください" }]
}
```

`details` はフォームバリデーションエラー時のみ付与する（任意項目）。

## コード値定義

### 性別コード（GENDER_CODE）

| キー   | コード | 表示名 |
| ------ | ------ | ------ |
| MALE   | 1      | 男性   |
| FEMALE | 2      | 女性   |
| OTHER  | 9      | その他 |

### 続柄コード（RELATIONSHIP_CODE）

| キー   | コード | 表示名 |
| ------ | ------ | ------ |
| SELF   | 1      | 本人   |
| SPOUSE | 2      | 配偶者 |
| CHILD  | 3      | 子     |
| PARENT | 4      | 親     |
| OTHER  | 5      | その他 |

### カテゴリタイプ（CATEGORY_TYPE）

| キー    | コード | 表示名 |
| ------- | ------ | ------ |
| EXPENSE | 1      | 支出   |
| INCOME  | 2      | 収入   |

### AI 機能コード（FEATURE_CODE）

| キー             | コード | 機能                                           |
| ---------------- | ------ | ---------------------------------------------- |
| RECEIPT_SCAN     | 1      | レシート読み取り・自動入力                     |
| EXPENSE_ANALYSIS | 2      | 簡易支出分析（ダッシュボード表示用）           |
| DETAILED_ADVICE  | 3      | 本格的アドバイス（地域・年齢・家族構成を考慮） |

### 頻度コード（FREQUENCY_TYPE）

| キー    | コード | 表示名 |
| ------- | ------ | ------ |
| MONTHLY | 1      | 毎月   |
| YEARLY  | 2      | 毎年   |

### 都道府県コード（REGION_CODE）

1〜47 の整数で表現。詳細は `packages/common/src/ui-constant.ts` の `REGION_MAP` を参照。

## バリデーション規則

| フィールド                    | 規則                           |
| ----------------------------- | ------------------------------ |
| カテゴリ名                    | 最大 50 文字                   |
| 表示名（family_members.name） | 最大 50 文字・必須             |
| メモ                          | 最大 255 文字                  |
| 誕生日                        | 日付（UNIX timestamp）         |
| 居住地                        | REGION_MAP のコード値（1〜47） |
